# PlantDiary — Design Context

## Audience
Home and hobby gardeners in Poland / Central Europe (climate zone 6a). Bilingual UI (Polish default, English). Used on phone in the garden and on desktop at the kitchen table.

## Use cases
Track gardens and individual plants, log watering/fertilizing/care, browse a large plant library (~14k) and rich bilingual plant profiles (care guide, diseases, photos), follow a seasonal calendar.

## Brand personality / tone
**Botanical atlas of the 16th century, rendered on an iPad.** A document with character and history — not a dashboard, not a generic app. Refined, editorial, warm. Zero emoji.

## Design system (AUTHORITATIVE — see frontend/design-reference/plant-detail.html)
This direction was specified by the user with a full HTML mockup. Match it.
- **Palette (CSS vars)**: parchment `#F7F3EC` background (+mid `#EDE7DB`, dark `#DDD4C5`); ink `#1C1C1A` (muted `#6B6459`, ghost `#A89F94`); copper accent `#B5672A` (light `#D4894F`, pale `#F5E8D8`) — the ONLY accent; forest `#1A2E1E` (mid `#2D4A31`, light `#3D6342`, pale `#E8F0E9`) for dark surfaces/CTA; warn `#8B4513`. radius 3px / 6px. White `#fff` cards on parchment, hairline `--parchment-dark` borders.
- **Type**: Playfair Display (serif display — headings, hero title, section titles, stat values, tip numbers, logo) + Inter (300/400/500/600 body/UI). Self-host via @fontsource.
- **Signature elements**: fixed blurred top nav w/ breadcrumb; dark forest hero w/ botanical SVG line-art + overlay + uppercase tags + huge Playfair title + difficulty; copper eyebrow (uppercase, letter-spaced) above each Playfair section title; copper-rule gradient dividers; drop-cap first paragraph; quick-stats bar; 2-col care-grid w/ icon tiles; botanical ornament divider (concentric-circle SVG); season timeline (month header row + active/harvest cells + copper "today" line + legend); problems list (badge + serif name + treatment box w/ forest-pale left border); companions grid (good/bad); serif-numbered tips; toxicity notice; sticky sidebar (forest CTA + forest-pale context card + params + pest tags + status).
- Icons: lucide-react line icons (no emoji). Motion subtle, no shadows on UI (only focus rings), no text gradients.
